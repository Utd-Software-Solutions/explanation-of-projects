🌎 المناطق الجغرافية

الشرق الأوسط — الدوحة

السيرفر الرئيسي — يشغّل Engine API + WebSocket Server + LiveKit Media Server. يخدم المنطقة العربية والشرق الأوسط وأفريقيا.

الهند — مومباي

سيرفر Relay — يشغّل LiveKit relay فقط لتقليل التأخير للمستخدمين في جنوب آسيا. لا يشغّل Engine API — كل الطلبات تمر عبر الدوحة.

ملاحظة: الدومين هو udt-stream.com (UDT وليس UTD في الدومين) — لأسباب تتعلق بتوفر النطاق.

🔄 كيف يعمل التوجيه

1

المطور يختار المنطقة

عند إنشاء المشروع، يحدد المنطقة الجغرافية الأقرب لمستخدميه.

2

Engine يولّد التوكن

التوكن يحمل عنوان سيرفر الميديا المناسب للمنطقة المختارة.

3

التطبيق يتصل تلقائياً

تطبيق المستخدم يقرأ عنوان السيرفر من التوكن ويتصل بالسيرفر الأقرب مباشرة.

🛡 الحماية — Cloudflare

DNS + SSL

Cloudflare أمام كل السيرفرات — يوفر DNS سريع + شهادات SSL/TLS تلقائية + حماية ضد هجمات DDoS.

عزل العملاء

بنية Multi-tenant — سيرفر واحد يخدم جميع العملاء مع عزل كامل بينهم. كل مشروع معزول على مستوى API والبيانات والغرف والإعدادات.
