محفظة Dev

المحفظة الفرعية لخدمة UTD Dev — منها يتم شراء الباكيجات والبلاجنز ودفع الباقات الشهرية

💳 كيف تعمل محفظة Dev؟

محفظة Dev هي محفظة فرعية تتبع المحفظة الأم — نفس نظام المحافظ الفرعية المستخدم في كل خدمات UTD.

الشحن والسحب

  • الشحن من المحفظة الأم فقط — تحويل فوري
  • إرجاع الرصيد للمحفظة الأم في أي وقت
  • لا يمكن شحن محفظة Dev مباشرة — لازم عبر المحفظة الأم
  • الحد الأدنى للشحن يحدده الأدمن

ما يُخصم من محفظة Dev

  • شراء باكيجات وبلاجنز مدفوعة من المتجر
  • الاشتراك في الباقات الشهرية (تجديد تلقائي)
  • شراء أحداث موسمية
  • أي خدمة مدفوعة داخل UTD Dev
القاعدة: المحفظة الأم ← محفظة Dev ← المتجر والباقات. نفس نظام المحافظ في باقي خدمات UTD — الفلوس تنزل من الأم للفرعية، والفرعية تصرف على الخدمة.

سجل المعاملات

جدول يعرض كل المعاملات المالية:

  • التاريخ والوقت
  • نوع المعاملة (شحن / شراء باكج / اشتراك باقة / إرجاع)
  • المبلغ (+ شحن أو - خصم)
  • الرصيد بعد المعاملة
  • الوصف (اسم المنتج أو الباقة)

الإجراءات المتاحة

  • شحن: تحويل مبلغ من المحفظة الأم لمحفظة Dev
  • إرجاع: إرجاع رصيد من محفظة Dev للمحفظة الأم
  • تنبيهات: ضبط تنبيه عند انخفاض الرصيد تحت مبلغ محدد
  • تصدير: تصدير سجل المعاملات CSV/PDF

التنبيهات

العميل يقدر يضبط تنبيهات تلقائية:

  • تنبيه عند انخفاض الرصيد تحت مبلغ محدد
  • تنبيه قبل تجديد الباقة الشهرية
  • تنبيه عند عدم كفاية الرصيد لتجديد الباقة
