📄 الفواتير الشهرية

أول ساعة من كل شهر جديد، النظام يُصدر فاتورة تلقائية بكل استهلاك الشهر اللي فات. الفاتورة حقيقية وفعلية — مش تقديرية.

ملاحظة: الفاتورة الشهرية بنفس شكل صفحة الاستهلاك — جدول يومي بكل خدمة + إجمالي الشهر + توضيح مصدر الخصم (باقة أو محفظة). العميل يقدر يراجعها ويقارنها بالمحفظة والباقة.
لا توجد فواتير تقديرية. الفواتير في النظام كلها حقيقية وفعلية — تُصدر بعد انتهاء الشهر بالكامل وتعكس الاستهلاك الفعلي الذي حصل.

📑 ما تحتويه الفاتورة

كل فاتورة شهرية تتكون من الأجزاء التالية:

🏢

Header (ترويسة الفاتورة)

شعار UTD Stream + بيانات العميل + رقم الفاتورة + الشهر

📊

جدول يومي

نفس شكل صفحة الاستهلاك — كل يوم = صف، كل خدمة = عمود. جدول كامل بكل أيام الشهر.

💰

تفصيل التكلفة اليومية

لكل يوم: إجمالي التكلفة + المبلغ المخصوم من الباقة + المبلغ المخصوم من المحفظة.

📈

ملخص الشهر

إجمالي كل خدمة على حدة + الإجمالي الكلي للشهر كامل.

📦

معلومات الباقة

لو كان عند العميل باقة — الفاتورة توضح كم استُهلك منها خلال الشهر وكم تبقى.

📌

Footer (ذيل الفاتورة)

معلومات التواصل مع UTD + الشروط والأحكام.
