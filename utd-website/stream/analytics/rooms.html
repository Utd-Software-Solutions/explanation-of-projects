الغرف والجلسات

كل غرفة أو جلسة بث في التطبيق يقدر العميل يشوف تفاصيلها الكاملة — مين دخل، مين طلع، كم المدة، إيش صار.

ماذا يرى العميل؟

📋

قائمة الغرف والجلسات

كل غرفة تعرض المعلومات التالية:

  • معرّف الغرفة
  • نوعها (مكالمة / بث مباشر / غرفة صوتية)
  • وقت البداية والنهاية
  • عدد المشاركين
  • الحالة (نشطة / منتهية / فشلت)
🔎

تفاصيل غرفة محددة

العميل يضغط على أي غرفة ويشوف:

  • الجدول الزمني — مين دخل ومين طلع ومتى
  • نوع المشاركة (صوت / فيديو / مشاهد)
  • مدة كل مشارك
  • جودة الاتصال لكل مشارك

البحث والفلترة

🔍

بحث

العميل يقدر يبحث بمعرّف الغرفة أو معرّف المستخدم للوصول السريع لأي غرفة أو جلسة محددة.

🔧

فلاتر

فلترة النتائج حسب:

  • النوع — مكالمة / بث / غرفة صوتية
  • الفترة الزمنية — اليوم / الأسبوع / الشهر / مخصص
  • المشروع — لو العميل عنده أكثر من مشروع
